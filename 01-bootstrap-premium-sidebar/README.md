# Bootstrap Premium Collapsible Sidebar

A reusable, responsive sidebar component built using Bootstrap 5 and JavaScript.

## Features

- Smooth collapse animation
- Icon-only collapsed mode
- Active link highlighting
- Mobile off-canvas behavior
- localStorage persistence
- Modern dark UI design

## Tech Stack

- HTML5
- Bootstrap 5
- CSS3
- JavaScript (ES6)
- Bootstrap Icons

## How to Run

1. Clone repository
2. Open index.html in browser
